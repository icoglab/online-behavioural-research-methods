6.1.1 How to Complete the Exercises

  1. Open each JavaScript file (e.g., ex01.js) in your code editor
  2. Read the instructions provided as comments at the top of each file
  3. Write your code directly in the JavaScript file following the given instructions
  4. Save the file and refresh your webpage to see the results
  5. Check your output in the browserâ€™s JavaScript console (not the webpage itself)

Important: Your code output will appear in the JavaScript console, not on the webpage. Make sure you have the developer tools open to see your results.
